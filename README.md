# House Price Prediction

## Project Overview

This project predicts house prices using Machine Learning and the Boston Housing Dataset. A Flask web application is developed to allow users to enter housing features and get predicted house prices.

## Technologies Used

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML
- CSS
- Git
- GitHub

## Dataset Features

- CRIM
- ZN
- INDUS
- CHAS
- NOX
- RM
- AGE
- DIS
- RAD
- TAX
- PTRATIO
- B
- LSTAT

## Installation

1. Clone the repository
2. Install dependencies using requirements.txt
3. Run app.py
4. Open http://127.0.0.1:5000

## Output

The application predicts house prices based on user input features.
